# 🏆 EBKA Championship Telegram Bot

Telegram бот для регистрации на турниры по Rocket League с админ-панелью.

## 🚀 Запуск

### Railway (рекомендуется)
1. Добавьте переменные окружения в Railway:
   ```
   TOKEN=ваш_бот_токен
   ADMIN_ID=ваш_id
   CHANNEL_ID=@ebka_news
   CHANNEL_LINK=https://t.me/ebka_news
   GROUP_LINK=https://t.me/ebka_group
   TURSO_URL=libsql://your-db.turso.io
   TURSO_TOKEN=your_turso_token
   ```

2. Деплой через GitHub

### Локально
```bash
pip install -r requirements.txt
export TOKEN="ваш_бот_токен"
python bot.py
```

## 🎮 Функционал

### Для пользователей
- 📝 **Регистрация на турниры**
  - Проверка подписки на канал
  - Ввод Epic ID, Discord
  - Выбор актуального и пикового MMR (через меню или вручную)
  - Ссылка на RL Tracker
- 📋 **Просмотр своих данных**
- 🗑 **Удаление своих данных**
- 💬 **Обращение к админам** (двухэтапный процесс)
- 🎮 **Discord** - ссылка на сервер

### Для администраторов
- ⚙️ **Админ-панель**
  - 📋 Список участников (полный)
  - 🎮 Удобный список игроков (красиво оформленный)
  - 📊 Статистика (количество участников)
  - 🗑 Удалить игрока по ID
  - 💥 Удалить всех участников

- 👥 **Управление админами**
  - ➕ Добавить админа по ID
  - 🗑 Удалить админа по ID
  - 📋 Список администраторов

- 📢 **Настройки канала**
  - ✏️ Изменить ссылку канала
  - 🎮 Изменить Discord ссылку
  - 🔔 Включить/выключить требование подписки
  - 👀 Посмотреть текущие настройки

- 🏆 **Управление турнирами**
  - ➕ Создать турнир (название, описание, дата, игроки, приз)
  - 📋 Список турниров
  - 🔔 Настроить уведомления

- 📝 **Приветственное сообщение**
  - ✏️ Изменить сообщение (отображается при /start)
  - 👀 Посмотреть текущее

- 📢 **Рассылки**
  - ➕ Создать рассылку (выбор турнира, сообщение, время)
  - 📋 Отложенные рассылки

### SuperUser (секретное меню)
- 🔐 Доступ по команде `/superuser` и паролю (по умолчанию: `1234`)
- 📊 Логи активности пользователей
- 📝 Логи бота
- 👥 Все пользователи
- 💾 Создать бэкап (JSON файл)
- 📂 Восстановить из бэкапа
- 🔑 Сменить пароль
- 🧹 Очистить логи
- 📈 Общая статистика

## 🗄️ База данных

### Turso (рекомендуется)
Облачная база данных для постоянного хранения. Данные сохраняются при редеплое.

### SQLite (fallback)
Локальная база данных. Данные могут удаляться при редеплое.

### Бэкапы
- Автоматическое создание JSON бэкапа через SuperUser меню
- Восстановление из файла бэкапа
- Включает: пользователей, админов, турниры, настройки, логи

## 📝 Переменные окружения

| Переменная | Описание | Обязательно |
|------------|----------|-------------|
| `TOKEN` | Токен бота от @BotFather | ✅ |
| `ADMIN_ID` | ID главного админа | ✅ |
| `CHANNEL_ID` | ID канала для подписки | ✅ |
| `CHANNEL_LINK` | Ссылка на канал | ❌ |
| `GROUP_LINK` | Ссылка на группу | ❌ |
| `TURSO_URL` | URL Turso базы данных | ❌ |
| `TURSO_TOKEN` | Токен Turso базы данных | ❌ |

## 🔧 Техническое

- **Python 3.11**
- **aiogram** - Telegram фреймворк
- **aiosqlite** - Асинхронная SQLite
- **libsql_experimental** - Turso клиент
- **aiohttp** - Веб-сервер
- **openpyxl** - Экспорт в Excel
- **Railway** - Деплой платформа

## 🌐 Веб-панель

Доступна на порту 5000:
- `/participants` - HTML таблица участников
- `/export` - Скачать Excel файл
- `/health` - Health check
